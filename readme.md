#365admin Spiderman

Tool for scrapping stuff our of homepages that we like to react to ...

## Installation

    npm install 
    npm install webdriver-manager -g

## Execution

In a console windows run 

    webdriver-manager start

Then in another

    node ./index.js crawl

or

    node ./index.js parse


## Verbose logging

    SET DEBUG=*
